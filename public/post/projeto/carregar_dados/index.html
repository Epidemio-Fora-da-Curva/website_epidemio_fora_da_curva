Carregar Dados no RStudio

Mar 17, 2020 5 minutos de leitura

Intro

Se você veio até aqui, você deve estar curioso em como carregar os dados no R ou no RStudio para poder iniciar sua análise de dados.

Não se preocupe! Neste post, eu irei desmotrar quatro maneiras simples que você pode utilizar para carregar dados que estão organizados em planilhas.

Mãos a obra

Carregar a partir de aquivo .txt

Utilize a função read.table(), a qual é uma função nativa do R para carregar dados que foram organizados ou gerados em um arquivo de texto ou bloco de notas. Normalmente esse tipo de arquivo possui extensão .txt.

read.table(file = "data.txt", header = T)
